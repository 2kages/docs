## 1. Introduction to the Agent Program

Vipro Studio's agent program is designed to support partners who want to distribute Vipro products in large quantities. Agents will enjoy special benefits:

- Higher discounts than the affiliate marketing program
- Ability to purchase accounts in bulk
- Dedicated agent management account
- Priority technical support
- Product training

## 2. How to Purchase Agent Package

1. Access [Vipro Profile](https://app.vipro.studio/profile)
2. Ensure that the language used on Vipro is Vietnamese. If Vipro's language is not Vietnamese, click on the flag icon in the left Toolbar

<img
  src='/en/images/partners/switch-language.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

<Warning>
  The Agent Program is only visible when you use Vietnamese language.
</Warning>

3. Select Become an Agent

<img
  src='/en/images/partners/become-agent.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

4. Purchase the Agent Package like [purchasing other products](/payment/check#2-purchase-product-package)

5. After the Agent Package has been purchased, in your Profile page, there will be an additional Order Management category. Select this category to access order creation and management

<img
  src='/en/images/partners/choose-agent-orders.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

## 3. Managing Agent Orders

- To create a new order, select Create New

<img
  src='/en/images/partners/create-agent-order.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

- Then, select the product package and duration you want to create, and click Create.
  If the agent's diamonds are insufficient to create the product package, that package will be grayed out.

<img
  src='/en/images/partners/choose-agent-package.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

- After a product package is created, the agent can copy the activation code for the buyer.

<img
  src='/en/images/partners/view-agent-credit.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

- If a product package hasn't been activated, the agent can recall the product.

<img
  src='/en/images/partners/delete-agent-order.jpeg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

<Note>
  Agents can only recall products when the product package hasn't been
  activated.
</Note>
