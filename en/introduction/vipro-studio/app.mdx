---
title: Using Vipro Studio on PC
description: "Guide to using Vipro Studio on PC"
---

To access Vipro Studio on the web platform, you can visit [https://vipro.studio](https://vipro.studio). Then follow these steps:

1. Click the **Try Unlimited** button to register a new account on the Vipro Studio system.

<img
  src='/en/images/introduction/home.jpg'
  alt='Vipro Studio on web'
  className='rounded-lg'
/>

2. Use your Gmail account to register

<img
  src='/en/images/introduction/login.jpg'
  alt='Register account with Gmail'
  className='rounded-lg'
/>

3. After registration is complete, you will be taken to the Vipro Studio dashboard. Here, choose the version suitable for your computer

<img
  src='/en/images/introduction/dashboard-app.jpg'
  alt='Vipro Studio Dashboard'
  className='rounded-lg'
/>

- If you are using Windows operating system, choose the Windows version
- If you are using a Mac computer from 2020 or earlier, choose the Intel Mac version
- If you are using a Mac computer from 2020 or later, choose the Mac M1 version

4. Download the file to your personal computer then extract the file and install the app if necessary

**For Windows computers:**

You need to install the software then open the file to use

<img
  src='/en/images/introduction/app-windows-1.jpg'
  alt='Vipro Studio App on Windows'
  className='rounded-lg'
/>

Remember to create a shortcut for convenient use

<img
  src='/en/images/introduction/app-windows-2.jpg'
  alt='Vipro Studio App on Windows'
  className='rounded-lg'
/>

**For Mac computers:**

You only need to extract and open the file to use

<img
  src='/en/images/introduction/app-mac.jpg'
  alt='Vipro Studio App on Mac'
  className='rounded-lg'
/>

5. After installation is complete, open the app and start logging in to the Vipro Studio system

When you open the app, Vipro Studio will ask you to log in

<img
  src='/en/images/introduction/app-login.jpg'
  alt='Log in to Vipro Studio'
  className='rounded-lg'
/>

Vipro will automatically open a new tab on your web browser to log in

<img
  src='/en/images/introduction/app-login-web.jpg'
  alt='Vipro Studio Dashboard'
  className='rounded-lg'
/>

Click on open application and you will be redirected back to your app

<img
  src='/en/images/introduction/app-success.jpg'
  alt='Vipro Studio Dashboard'
  className='rounded-lg'
/>

After logging in to the system, you will be given 60 diamonds to use to create videos for free
