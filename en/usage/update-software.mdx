### Check software version

Click on the Avatar in the bottom right corner, the software version will be displayed.

<img src="/en/images/usage/update-software/check-version.webp" alt="Check Version" className="rounded-lg" />

### View version changelog

Select "Software History" to access the list of changes for each version.

<img src="/en/images/usage/update-software/history-version.webp" alt="History Version" className="rounded-lg" />

### Manually update to a new version

Click on the Avatar in the bottom right corner, select "Check for updates" to update to the latest version.

<img src="/en/images/usage/update-software/check-update.webp" alt="Check Update" className="rounded-lg" />

When a new version is available, a notification will be displayed on the interface. Select "Go to download page" to download the latest version.

<img src="/en/images/usage/update-software/update-dialog.webp" alt="Update Dialog" className="rounded-lg" />

On the download page, depending on whether your computer is Mac or Windows, select the corresponding version to download.

<img src="/en/images/usage/update-software/download-version.webp" alt="Download Version" className="rounded-lg" />
